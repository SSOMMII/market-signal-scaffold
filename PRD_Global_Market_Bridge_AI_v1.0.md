# [PRD] AI 기반 글로벌 주식 시장 연결 예측 서비스 (v1.0)

- 작성일: 2026년 3월 25일
- 문서 상태: 초안 (Draft)
- 프로젝트명: Global Market Bridge AI (가칭)

## 1. 서비스 개요 (Service Overview)

본 서비스는 국내(국장)와 미국(미장) 주식 및 ETF 데이터를 통합 분석하여, 시장 간의 시차를 이용한 최적의 투자 시나리오와 시그널을 제공하는 AI 플랫폼입니다.

- 핵심 가치: "잠들지 않는 시장의 연결" - 미국장 종료 후 국장 대응, 국장 종료 후 미국장 대응 전략 수립
- 주요 타겟: 글로벌 거시 지표와 기술적 지표를 동시에 참고하여 ETF 투자를 즐기는 개인 투자자

| 사용자 체류시간 | 인당 평균 5분 이상 | 리포트 정독 및 지표 확인 시간 |

## 3. 정보 구조 (Information Architecture)

### 3.1 메인 구조 (Home)

- Night Insight (미장 → 국장): 미국 증시 결과 기반 익일 한국 시장 예측
- Day Insight (국장 → 미장): 한국 증시 결과 및 외국인 수급 기반 당일 미국 시장 예측
- Today's Signal: AI가 선별한 오늘의 Top ETF 추천
- Global Dashboard: S&P500 선물, 환율, VIX 등 핵심 지표 실시간 뷰어

### 3.2 상세 페이지 구성

- Insight Detail: 예측 근거(뉴스 요약, 지표 변화) 및 AI 생성 리포트
- ETF Signal Lab: 매수/매도/관망 시그널 및 백테스트 데이터 시각화
- My Portfolio: 관심 종목에 대한 AI 예측 알림 설정

## 4. 데이터 지표 매핑 (Data Mapping)

## 5. ETF 시그널 알고리즘 (Logic)

### 5.1 Signal Score 산출 공식

```text
Signal Score = (Tech * 0.3) + (Global * 0.3) + (Futures * 0.2) + (FX * 0.1) + (Supply * 0.1)
```

### 5.2 최종 시그널 기준

- +2점 이상: 🟢 STRONG BUY (공격적 매수)
- -2점 이하: 🔴 STRONG SELL (현금화 및 인버스 고려)
- 기타: 🟡 HOLD (관망 및 포지션 유지)

## 6. 기술 사양 및 DB 설계 (Technical Spec)

### 6.1 데이터베이스 테이블 (Key Tables)

- `market_master`: 종목 및 시장 기본 정보
- `daily_indicators`: 기술적 지표 저장 테이블
- `ai_signals`: AI가 생성한 최종 스코어 및 예측 결과
- `user_alerts`: 사용자 맞춤 푸시 알림 설정

### 6.2 연동 API

- 국내: 한국투자증권 API (실시간 시세 및 수급)
- 해외: Alpha Vantage, Yahoo Finance, Finnhub
- 거시: FRED (금리 및 주요 경제 지표)

## 7. Python 예측 모델 프로토타입 (Logic Check)

```python
# 핵심 로직 요약
def calculate_signal(row):
    score = 0
    if row['rsi'] < 30: score += 1  # 과매도 구간 매수 가점
    if row['macd'] > 0: score += 1  # 상승 추세 가점
    if row['futures'] > 0: score += 1 # 선물 시장 긍정
    if row['usd_krw'] < 1300: score += 1 # 환율 안정 (외인 수급 긍정)
    return score
```

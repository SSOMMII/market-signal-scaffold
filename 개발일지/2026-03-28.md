### 2. Confidence Threshold 도입

**파일: `app/api/signals/route.ts`, `app/api/signals/top/route.ts`**

LightGBM 신뢰도 기반 신호 필터링으로 거짓 신호 감소.

```typescript
// 신뢰도 >= 0.60: 신호 유지
// 신뢰도 < 0.60: 관망으로 다운그레이드
// 신뢰도 = null: 기술적 스코어만 사용
```

매수/매도 기준 동일하게 대칭 처리. 응답 필드: `confidence` (0-1, null 가능)

---

### 3. 신호 강도 5분류

**파일: `app/api/signals/route.ts`, `app/api/signals/top/route.ts`, `app/page.tsx`, `lib/marketData.ts`**

기존 3분류(매수/관망/매도) → 5분류(강도 명시)로 진화.

| 신호 | 조건 |
|------|------|
| 🚀 강한 매수 | score ≥ 75 + confidence ≥ 0.60 |
| 📈 매수 | score ≥ 60 |
| ➡️ 관망 | 40 ≤ score < 60 |
| 📉 매도 | 20 ≤ score < 40 |
| 🔴 강한 매도 | score < 20 + confidence ≥ 0.60 |

배지 색상: 강도별 5단계 시각화 (진한 초록 → 연한 초록 → 회색 → 연한 빨강 → 진한 빨강)

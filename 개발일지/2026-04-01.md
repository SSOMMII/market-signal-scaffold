## 2. 수정된 주요 버그

### 1) 예측이력 API 정확도 통계 로직 개선

- **파일**: `app/api/history/route.ts`
- **문제**: 데이터 부재 시 `actual = direction`으로 강제 설정되어 적중률이 과도하게 높게 표시되던 오류.
- **수정**: `actual`, `hit` 타입을 `null` 허용으로 확장하고, 미검증 레코드는 통계 계산 분모에서 제외하여 데이터 정확도 확보.

### 2) 예측이력 초기 데이터 노출 오류

- **파일**: `app/history/page.tsx`, `lib/historyData.ts`
- **문제**: API 응답 전 `mockPredictions`가 일시적으로 노출되던 문제.
- **수정**: 초기값을 빈 배열(`[]`)로 설정하고, 검증 대기 항목은 회색 `?` 배지로 표시.

### 3) 상세분석 페이지 인코딩 오류

- **파일**: `app/detail/page.tsx`
- **문제**: `signal === '매수'` 조건문이 인코딩 문제로 깨져서 항상 `false`로 작동하던 오류.
- **수정**: 해당 로직을 정상적인 UTF-8 텍스트로 교체 완료.

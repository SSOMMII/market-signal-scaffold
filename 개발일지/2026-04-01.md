KIS_APP_KEY / KIS_APP_SECRET이 `.env.local`에 설정되어 있으므로 아래 절차로 확인한다.



**문제 응답 (KIS API 미지원 시):**

| 오류 | 원인 | 대응 |

### Step 3. KIS 계정이 investor flow 미지원인 경우 대안

**KIS 무료 계정(모의투자/개인)** 은 대부분 `inquire-investor` tr_id를 지원하지 않음.
이 경우 KRX Open API로 전환이 필요하다.

#### KRX Open API 사용 방법 (무료)
1. [KRX 정보데이터시스템](https://data.krx.co.kr) 회원가입
2. Open API 신청 → API Key 발급 (승인 1~3 영업일)
3. `.env.local`에 `KRX_API_KEY=발급받은키` 추가
4. `/api/cron/kis-flow/route.ts` 또는 `/api/foreign-flow/route.ts`에 KRX endpoint 호출 추가:
   ```
   GET https://data.krx.co.kr/comm/bldAttendant/getJsonData.cmd
     bld=dbms/MDC/STAT/standard/MDCSTAT02301
     indIdx=&indIdx2=&itmCd=&itmCd2=&tboxindTpCd_finder_secuprodisu1_0=&isuCd=&isuNm=&indTpCd=&isuCd2=&trd_dd=20260401&share=1&money=3
   ```
   → KOSPI 외국인 순매수 데이터 포함
